## Инструкции к каждой задаче

Каждая папка в репозитории представляет собой отдельное задание. Внутри каждой папки находится файл с исходным кодом решения (`task_X.go`), где X - номер задачи.

## Запуск

Для запуска любой из задач, выполните следующие шаги:

1. Перейдите в папку с задачей: `cd taskX`, где `X` - номер задачи.
2. Запустите программу: `go run taskX.go`.
